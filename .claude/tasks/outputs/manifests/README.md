**Location**: `.claude/tasks/outputs/manifests/`

## How to Use Across Sessions

### For a New Conversation

When starting a new AI session and want full context:

1. **Quick onboarding**: Share the relevant manifest(s) based on your task:
   - Architectural questions → `TECHNICAL_MANIFEST.md`
   - Code changes → `CODE_MANIFEST.md`
   - Infrastructure changes → `TERRAFORM_MANIFEST.md`

2. **Prompt template**:
   ```
   I'm working on [task description]. Here's the project context:
   
   [paste relevant manifest(s)]
   
   [your specific question or request]
   ```

3. **Saves time**: AI has full context without re-reading 100+ files

### For Maintaining Manifests

**When to update**:
- After adding new modules or services
- After changing deployment procedures
- After refactoring major code sections
- After updating dependencies
- Every 1–3 months (to stay current)

**How to update**:
1. Use the corresponding `TASK_GENERATE_*.md` file in `.claude/tasks/` as a guide
2. Run through the process phases (explore, map, extract, generate)
3. Update the manifest with new findings
4. Save updated manifest to `.claude/tasks/outputs/manifests/`
5. Commit updated manifests to git

**Tip**: Mark outdated sections with `[OUTDATED — last verified YYYY-MM-DD]` so readers know when to verify.

## Tips for Effective Use

1. **Keep manifests current**: Stale docs are worse than no docs. Add a "last updated" date at the top of each manifest.

2. **Link to source**: Include file paths and line numbers so readers can verify claims against actual code.

3. **Use examples**: Real code excerpts and walkthroughs beat abstract descriptions.

4. **Document tradeoffs**: Explain *why* a design decision was made, not just *what* was decided.

5. **Highlight gotchas**: Non-obvious behaviors and common mistakes save debugging time.

6. **Version the manifests**: If you maintain multiple versions of the system (e.g., v1, v2), version your manifests too.

If a manifest is incomplete, outdated, or unclear:

These manifests are living documents — keep them accurate and they'll save hours across sessions.
